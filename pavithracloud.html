Cloud Computing

by Pavithra

Cloud computing is used for storing and accessing data and programs over the internet instead of computer ‘s hard drive. There are boardly classified into types , they are Public cloud and Private cloud. Public cloud refers to a common cloud service and it is available to multiple subscribers. The cloud service are owned and operated by a third- person service provider and delivered over the internet. It has the same hardware, storage and network devices with other organizations which use the same cloud, called cloud “tenants”. Examples Amazon Cloud Drive, icloud, etc….. Private cloud consists of computing resources used exclusively owned by one business or organization. It is often used by government agencies, financial institutions, any other mid- to large-size organizations. Private cloud also known as “internal” or “enterprise” cloud. This data are behind by a firewall. Advantages are less costs, flexibility in capacity, automated updates, security, control on the documents, easily manageable etc… so now a days it has been used widely.
